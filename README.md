# SSS-X1-Logics

## Deutsch

Dieses Repository ist das öffentliche Release- und Downloadrepository für fertige Gira-X1-Logikbausteine von Smart System Solutions.

Der Quellcode liegt nicht in diesem Repository. Die ZIP-Dateien sind fertige Artefakte für Anwenderinnen und Anwender.

Alle Inhalte in diesem Repository sind als öffentlich sichtbar zu behandeln. Interne Entwicklungsdetails, lokale Pfade, private Repository-Namen, Lizenzschlüssel und nicht freigegebene Vertriebsinformationen gehören nicht in dieses Repository.

### Inhalt

- Fertige ZIP-Dateien pro Logikbaustein
- Mehrere Versionen je Baustein, soweit vorhanden
- Eine zentrale Übersicht in [BAUSTEINE.md](BAUSTEINE.md)
- Lizenz- und Drittanbieterhinweise innerhalb der ZIP-Dateien

### Struktur

Jeder Baustein liegt in einem eigenen Ordner. Die ZIP-Dateien enthalten aktuell Versionsnummern im Dateinamen.

Beispiel:

```text
AccuManage/
  smart_system_solutions_de.Logic.AccuManage-3.0.24.zip
```

### Downloads

Für Website-Downloads sollte auf konkrete ZIP-Dateien verlinkt werden. Eine zentrale Liste der aktuell erkennbaren Versionen steht in [BAUSTEINE.md](BAUSTEINE.md).

Vor der finalen Veröffentlichung von Downloadlinks sollte geprüft werden, ob die Pfade dauerhaft stabil bleiben sollen.

### Lizenzstatus

Die technischen Manifeste der geprüften ZIP-Dateien enthalten aktuell `License: Free`. Gleichzeitig enthalten die mitgelieferten Lizenztexte allgemeine Formulierungen zu Kauf und Lizenzschlüsseln.

Deshalb wird der öffentliche Vertriebsstatus in diesem Repository vorerst als `noch zu prüfen` geführt. Lizenzinformationen werden nicht geraten.

## English

This repository is the public release and download repository for finished Gira X1 logic modules by Smart System Solutions.

The source code is not stored in this repository. The ZIP files are finished artifacts for users.

All content in this repository should be treated as publicly visible. Internal development details, local paths, private repository names, license keys, and non-approved distribution information do not belong here.

### Contents

- Finished ZIP files per logic module
- Multiple versions per module where available
- A central overview in [BAUSTEINE.md](BAUSTEINE.md)
- License and third-party notices inside the ZIP files

### Structure

Each module is stored in its own folder. The ZIP files currently include version numbers in their filenames.

Example:

```text
AccuManage/
  smart_system_solutions_de.Logic.AccuManage-3.0.24.zip
```

### Downloads

Website downloads should link to concrete ZIP files. A central list of currently detected versions is available in [BAUSTEINE.md](BAUSTEINE.md).

Before publishing final download links, the paths should be confirmed as stable.

### License Status

The technical manifests of the checked ZIP files currently contain `License: Free`. At the same time, the included license texts contain general wording about purchase and license keys.

For that reason, the public distribution status in this repository is currently listed as `noch zu prüfen` / `to be checked`. License information is not guessed.
